# threejs-boilerplate

Simple project set up for three.js experiments

Built with:

- Webpack
- Babel
- SASS

### Commands

Setup using:

```bash
yarn install
```

Start a development server with watch tasks:

```bash
yarn start
```

Build for production:

```bash
yarn build
```
